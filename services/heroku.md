# Heroku

## Environment variables

List variables

```bash
heroku config
```

Set an environment variable

```bash
heroku config:set <key>="<value>"
```

get an environment variable

```bash
heroku config:get <key>
```

Remove an environment variable

```bash
heroku config:unset <key>
```
